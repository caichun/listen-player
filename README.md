## 🎯 核心播放复读功能（左侧控制栏）

### 🔁 A‑B 精细化分段复读
- 手动设置 **A 起点 / B 终点**，一键锁定区间循环播放  
- **「整段」快捷按钮**：快速锁定当前字幕对应片段，精准截取单句/段落  
- 底部实时显示 A‑B 时间轴，**波形同步标记**区间范围，可视化定位  
- 自定义 **回退时长**（默认 5s）、**无限复读次数**，自动循环反复听单句

### ⏯️ 基础播放快捷操控
- 播放/暂停 · 5s 快进/快退 · 重置进度  
- **变速播放**：0.5× ~ 2.0× 可调（当前 1×）  
- ⌨️ **全功能快捷键**：`空格` 启停，`←`/`→` 进退，`A`/`B` 设区间 —— 彻底脱离鼠标

### 📊 可视化波形进度条
- 音频波形图实时渲染，**声音高低一目了然**  
- 不用听就能定位 **人声、停顿、空白片段**，快速跳过杂音，锁定有效听力内容

## 📝 字幕同步系统（右侧字幕栏）

- **时间轴精准对齐**：每条字幕绑定精确 `HH:MM:SS` 时间戳，播放时自动高亮当前句  
- **轻量化管理**：独立导入 `.srt`/`.ass` 等字幕文件，一键 **隐藏/显示**，支持盲听训练  
- **字幕与复读联动**：点击任意字幕时间片段 → 播放器自动跳转并 **自动设定 A‑B 区间**，零成本截取

## ✍️ 内置听写训练模块（中音视频下方）

> 区别于普通播放器的 **核心学习功能**，实现“听 + 写”闭环：

- 同步抓取当前播放字幕句子，下方输入框 **实时听写**  
- 配套按钮：`显示答案` · `下一句` · `清空输入`  
- **听写模式** 开关：开启后自动隐藏字幕，强制盲听默写，提升实战听力
